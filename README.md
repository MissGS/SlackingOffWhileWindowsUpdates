# SlackingOffWhileWindowsUpdates

**让你想偷懒的时候，领导或者老师无话可说 😎**

## 简介
`SlackingOffWhileWindowsUpdates` 灵感来源于每个人都经历过的神奇时刻——**电脑在更新的时候，你终于可以偷个懒而不用被发现**。  
这是一个带点幽默、轻松的项目/概念，适合用来记录生活小技巧、搞笑脚本或者懒人工具。

## 特性
- 💻 充分利用 Windows 更新时间偷懒
- 😂 让你偷懒也显得合理
- 📝 适合做幽默项目或者轻量工具合集

## 用法
只要你的电脑在更新，就可以：
- 喝杯咖啡 ☕  
- 拉个小瞌睡 😴  
- 或者做自己真正想做的事情 🎨  

> 总之，**Windows 更新 = 偷懒正当理由**
